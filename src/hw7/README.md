<h1>Задание 3 (2 балла)</h1>

<p>-Вы, Сэры, преступники, а сидеть Вам следует не в казино, а в тюрьме - заявил Микки Роса - я предлагаю заработать честным образом. Увеличиваем или уменьшаем счёт, находим максимальное значение, достигнутой за день, а уже после этого приходим в казино и спокойно заходим каждый раз, когда счёт доходит до обнаруженного нами максимального значения.</p>

<p>-Счёт? О чём ты, Микки?</p>

<p>-Джентельмены, в этой игре выгодно ставить, когда из колоды убраны определённые карты. Для этого математики придумали стратегию, по которой каждая вышедшая из колоды карта изменяет значение счётчика. Сколько каждая карта добавляет или убавляет от текущего значения, которое начинается с нуля в начале игрового дня, я вам расскажу позже. В нашем Лас Вегасе играют по-честному, а потому колоду не меняют, пока в ней ещё есть карты. Для игры используют одну обычную колоду карт.</p>

<p><strong>Задание</strong>: По таблице:</p>

<pre><code>create table TwentyOneTables (
 id int,
 table_n int, -- table number
 card varchar(2) -- type of card: like ♥4 or ♠J
);
</code></pre>

<p>Найдите максимальное значение <q>счётчика</q> за шесть игр. В этой задаче считается, что атрибут с номером стола означает номер игры. В начале каждой игры счётчик обнуляется.</p>

<p>Изменение счётчика:</p>

<ul>
<li><p>Валет, Дама, Король и Туз уменьшают счётчик на 1</p></li>
<li><p>3, 4, 6 увеличивают счётчик на 1</p></li>
<li><p>5 увеличивает счётчик на 1.5</p></li>
<li><p>9 уменьшает счётчик на 0.5</p></li>
<li><p>10 (то есть 0) уменьшает счетчик на 1</p></li>
<li><p>2 и 7 увеличивают счётчик на 0.5</p></li>
<li><p>8 не изменяет значение счётчика</p></li>
</ul>
